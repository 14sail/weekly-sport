# 每週運動排程系統

給 LINE 群組使用的每週運動時間協調小工具。

🔗 使用網址：https://14sail.github.io/weekly-sport/

## 功能

- 成員回報未來 4 週、每天的空閒狀態（有空／不確定／沒空），並可留言給老師
- 老師檢視每週統計、取得建議日期、產生公告文字，一鍵複製貼到 LINE 群組
- 管理入口可新增／刪除成員、指定或取消老師身份

## 技術架構

單一 `index.html`（原生 JavaScript），資料庫使用 Firebase Realtime Database，部署於 GitHub Pages。

## 使用方式

1. 開啟網站，選擇自己的名字
2. 第一次使用需設定密碼，之後用密碼登入
3. 成員：勾選每天的空閒狀態並儲存
4. 老師：切換到「老師管理視角」查看統計、產生公告
